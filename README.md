## 🚀 Setting Up the AI

VibeHarness relies on a tool called Ollama to manage the model. You must install this tool before you run the harness. 

1. Go to the Ollama website and download the Windows installer.
2. Run the installer and click finish when it completes.
3. Open your command prompt by pressing the Windows key, typing cmd, and pressing enter.
4. Type `ollama run vibethinker` into the black window and press Enter.
5. Wait for the green progress bars to finish loading the model files.
6. Once the text stops moving, press Ctrl+D to exit this specific window.

## 🖱️ Running the Program

After you finish the setup, you launch VibeHarness directly from your desktop icon. 

1. Double-click the VibeHarness shortcut. 
2. A window opens and shows that the agent is ready.
3. Type your instructions into the box provided. 
4. Press the Enter key to send your request.
5. The small agent will process your command and return an answer. 
6. Keep the window open as long as you need assistance.

## 🧠 What You Can Do

The agent supports several tasks to help your workflow. You can ask for summaries of long text files. You can request help with basic coding questions or language translation. The agent performs these tasks without sending your information to the cloud. Everything happens inside your computer. 

## 🛠️ Troubleshooting

If the program fails to start, check if your antivirus software blocked it. Many security tools look at new programs with caution. Add an exception for VibeHarness in your settings if you see a warning. 

Make sure you closed other heavy programs like video editors or games before you launch the agent. These programs fight for the memory that the AI model needs. 

If the model response feels slow, restart your computer to clear the memory. Occasionally, Windows keeps processes running even after you close them. A fresh start solves most lag issues. 

If the agent returns an error, verify that Ollama stays active in your system tray. Look for the small icon near your clock at the bottom right of the screen. If you do not see the icon, click your start menu, find Ollama, and launch it again.
